Quick Start Guide
1. Use your API key in requests:

     {{ request.url.scheme }}://{{ request.url.netloc }}/movies

curl -H "X-API-KEY: YOUR_API_KEY" \
     "{{ request.url.scheme }}://{{ request.url.netloc }}/search?title=batman"
